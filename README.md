# python-五子棋-b站直播用

## 使用方法
 将main.py内的roomID改成你的房间号就可以了。
 
 默认设置：
 
 黑棋：全体观众（统计一分钟内的弹幕）
 
 白棋：主播（鼠标点击）
 
 修改StateMachine类的white和black常量可以将黑棋和白棋设置为：任意观众（'any'），全体观众（'any'），特定观众（观众的昵称），ai（'ai'），主播（'mouse'）。
